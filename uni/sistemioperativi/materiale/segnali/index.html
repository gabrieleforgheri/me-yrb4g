Segnali in C

[Context: Comunicazione asincrona tra processi tramite interruzioni software generate dal Kernel o da altri processi.]

1. INVIARE SEGNALI

I segnali sono notifiche asincrone. Un processo può inviare un segnale a un altro processo usando la system call kill().

#include <signal.h>

// Invia SIGUSR1 al processo con PID specificato
if (kill(target_pid, SIGUSR1) == -1) {
    perror("kill fallita");
}

// alarm(N) invia un SIGALRM a se stesso dopo N secondi
alarm(5);

2. GESTIRE SEGNALI (Handlers)

Per definire come un processo reagisce a un segnale, si usa sigaction (preferibile alla vecchia signal()).

#include <signal.h>

// Variabile globale safe per i signal handler
volatile sig_atomic_t flag = 0;

void handler(int sig) {
    if (sig == SIGUSR1) {
        flag = 1;
    }
}

int main() {
    struct sigaction sa;
    sa.sa_handler = handler; // Assegna la funzione handler
    sa.sa_flags = 0;         // Flag addizionali (es. SA_RESTART)
    sigemptyset(&sa.sa_mask); // Maschera vuota durante l'esecuzione dell'handler

    if (sigaction(SIGUSR1, &sa, NULL) == -1) {
        perror("sigaction");
    }
    
    while(!flag) {
        pause(); // Mette in sleep il processo fino all'arrivo di un segnale
    }
}

3. MASCHERE E SIGNAL SETS

Spesso è necessario bloccare momentaneamente alcuni segnali in sezioni critiche del codice. I segnali bloccati restano "pendenti" e vengono consegnati appena sbloccati.

sigset_t set;
sigemptyset(&set);          // Inizializza il set vuoto
sigaddset(&set, SIGINT);    // Aggiunge SIGINT al set

// Blocca i segnali presenti nel set
sigprocmask(SIG_BLOCK, &set, NULL);

/* --- SEZIONE CRITICA --- */

// Sblocca i segnali
sigprocmask(SIG_UNBLOCK, &set, NULL);

4. TABELLA SEGNALI COMUNI

Segnale Azione Default Descrizione
SIGINT (2) Termina Generato da CTRL+C da tastiera. Intercettabile.
SIGKILL (9) Termina (Forzato) Uccisione immediata. Non può essere intercettato, bloccato o ignorato.
SIGSTOP (19) Sospende Mette in pausa il processo. Non può essere intercettato.
SIGALRM (14) Termina Generato alla scadenza di un timer impostato con alarm().
SIGUSR1 / SIGUSR2 Termina Segnali custom a disposizione del programmatore per la logica dell'app.
